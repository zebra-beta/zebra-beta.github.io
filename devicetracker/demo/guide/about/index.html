About Device Tracker Demo

Device Tracker (Demo)

FOR INTERNAL ZEBRA USE ONLY - DO NOT DISTRIBUTE


Device Tracker Demo contains the latest release of Device Tracker intended for Zebra SE use for customer demonstrations.

Overview

Device Tracker is a cloud-based scalable solution that makes it easy to track and recover lost or misplaced devices, preventing device inventory shrinkage. No additional hardware is required - leverage existing Wi-Fi network infrastructure along with Bluetooth and audio to locate devices. When locating a device, Device Tracker identifies the general vicinity of the lost device based on the Access Point (AP) it is connected to within the facility. The visual proximity indicator shows in real-time how close or far the lost device is located, relying on Bluetooth beacon transmissions. If the device is not within line-of-site, an alarm can be remotely triggered on the device to further determine its precise location.

Install & Setup

To install and configure Device Tracker client app:

  1. Connect the device to the internet via Wi-Fi.

  2. Launch the StageNow client.

  3. Scan the appropriate barcode based on region:

    • For NA, scan barcode.
    • For EMEA, scan barcode.
    • For APAC, scan barcode.

    This performs the following:
              1. Download and install Device Tracker
              2. Disable battery optimization
              3. Grant the required permissions to run Device Tracker
              4. Start the Device Tracker service
              5. Configure the server connectivity settings to communicate with the respective cloud server (based on region)

  4. The screen below appears while the server configuration is taking place (may take more than 10 seconds to complete) followed by the Checkout screen blocking device access. This confirms the server connection is established and the client app is ready for use.

         
    Server configuring after install.       Checkout block access screen.

  5. Scan any barcode to remove the Checkout screen. Checkin/checkout is enabled for the demo. The checkout screen appears each time the device is connected to external power.

  6. Login. Tap on the options menu at the top-right and select Admin/Manager Login. Enter your user credentials.

  7. Register your devices.

  8. Register your access points.
